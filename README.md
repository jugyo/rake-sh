rake-sh
====

The Rake Shell.

Install
----

    gem install rake-sh

Launch
----

    rake-sh

or specify eager tasks:

    rake-sh environment

Usage
----

Invoke a rake taks:

    rake> rake spec

Print rake tasks:

    rake> tasks

or specify a pattern:

    rake> tasks db
